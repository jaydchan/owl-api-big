# owl-api-big

## Requirements

* maven
* java (13) -> update pom file for desired version

## Usage

To create owl-api-big jar file (with dependencies) type `make`.

## Additional make commands

* o10.owx
* o100.owx
* o1000.owx
* o10000.owx
* o100000.owx
* o1000000.owx
* o2500000.owx
* o5000000.owx
* o10000000.owx
* o20000000.owx -> fails
* small -> ontologies 10^1 to 10^5
* large -> ontologies 10^6 to 2*10^7
